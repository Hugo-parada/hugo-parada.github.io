# Hugo Parada — Academic Homepage

This repository contains the source code for my academic website hosted with GitHub Pages.

## Website

The site is available at:

`https://hugo-parada.github.io`

## Contents

The homepage includes:

- About
- Research interests
- Selected publications and preprints
- Student advising
- Collaborators
- Juggling and outreach
- Contact information

## Editing the site

The main file is:

- `index.html`

To update the website:

1. Edit `index.html`
2. Commit the changes
3. Push to the `main` branch

GitHub Pages will publish the updated version automatically if the repository is configured to deploy from the root of the `main` branch.

## Notes

This website was adapted from a previous Google Sites page and simplified into a single-page academic homepage for GitHub Pages.
